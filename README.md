# FalseCam ✨

> 당신의 평범한 일상을 특별한 순간으로 재구성하는 AI 기반 추억 생성 서비스

FalseCam은 사용자가 입력한 짧은 텍스트와 사진 한 장을 분석해, 세상에 없던 독창적인 AI 브이로그(이미지/영상)를 만들어주는 웹 서비스입니다. 최신 생성형 AI 기술을 활용하여, 당신의 평범했던 순간을 영화처럼, 때로는 애니메이션처럼 특별하게 재구성해 보세요. 복잡한 프롬프트 작성이나 전문 지식 없이 누구나 쉽고 재미있게 자신만의 콘텐츠를 만들 수 있습니다.

## 🚀 주요 기능

* **AI 경험 재구성 (AI Experience Reconstruction)**: "카페에서 공부했다"와 같은 단순한 일상 기록을 AI가 "네온사인이 비치는 창가에서 홀로그램 노트북으로 코딩하는 사이버펑크 개발자"처럼, 전혀 다른 장르의 '있을법한 하루'로 새롭게 창조합니다.
* **자동 프롬프트 생성 (Automatic Prompt Generation)**: 복잡한 영어 프롬프트가 필요 없습니다. 사용자가 평범한 한글 문장을 입력하면, AI가 자동으로 장면, 구도, 분위기를 고려한 상세한 영어 프롬프트로 변환하여 이미지 생성 품질을 극대화합니다.
* **다양한 스타일 선택 (Various Style Options)**: `실사`, `2D 애니메이션`, `3D 애니메이션`, `사이버펑크`, `도트` 등 원하는 스타일을 직접 선택하여 결과물을 개인화할 수 있습니다.
* **이미지 to 비디오 변환 (Image-to-Video Conversion)**: AI가 생성한 이미지를 기반으로, 그 장면에 어울리는 자연스러운 움직임이 더해진 짧은 비디오 클립을 자동으로 생성합니다.
* **회원가입 없는 간편함 (Simplicity without Sign-up)**: 별도의 회원가입 절차 없이, 익명 세션(Session) 기반으로 즉시 서비스를 이용하고 연속적인 작업 흐름을 지원하여 사용자 편의성을 높였습니다.

## ⚙️ 아키텍처

FalseCam은 프론트엔드와 백엔드가 분리된 **Decoupled Architecture**를 채택하여 각 파트의 독립성과 확장성을 확보했습니다.

[Client (Browser)] <=> [Frontend (React on Cloudflare)] <=> [Backend (FastAPI on Render)] <=> [External AI Services (OpenAI, fal.ai)]

* **CI/CD**: `GitHub` 저장소의 `main` 브랜치에 코드가 푸시되면, `Cloudflare Pages`와 `Render`가 이를 감지하여 자동으로 빌드 및 배포를 진행하는 CI/CD 파이프라인을 구축했습니다.
* **AI Orchestration**: 백엔드는 사용자의 단일 요청을 받아, **LLM → Image → Video** 순서로 여러 AI 모델 API를 순차적으로 호출하고 데이터를 가공하여 최종 결과물을 반환하는 오케스트레이션(Orchestration) 역할을 수행합니다.
